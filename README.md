# i2cLCD2004

makecode I2C LCD2004 package for micro:bit  

Author: jieliang Mo  
Date:   2021.July  
  
![](lcd.jpg)

## Add extension

open your microbit makecode project, in Extension, paste  

https://github.com/makecode-extensions/i2cLCD1602  

to search box then search.

## Basic usage

```
let item = 0
I2C_LCD2004.LcdInit(0)
I2C_LCD2004.ShowString("Hello", 0, 0)
basic.forever(() => {
    item += 1
    I2C_LCD2004.ShowNumber(item, 0, 1)
    basic.pause(1000)
})
```


## I2C Address  
- PCF8574: 39  
- PCF8574A: 63  
- Auto: 0

## API

- LcdInit(Addr: number)  
Initial LCD  
Addr: I2C Address. If Addr is zero, it will try to recognition correctly address automaticly.  

- ShowNumber(n: number, x: number, y: number)  
show a number in LCD at given position.  
n: number will be show  
x: is LCD column position, [0 - 15]  
y: is LCD row position, [0 - 1]  

- ShowString(s: string, x: number, y: number)  
show a string in LCD at given position.  
s: string will be show  
x: is LCD column position, [0 - 15]  
y: is LCD row position, [0 - 1]  

- on()  
turn on LCD  

- off()  
turn off LCD  

- clear()  
clear LCD content  

- BacklightOn()  
turn on LCD backlight  

- BacklightOff()  
turn off LCD backlight  

- shl()
shift left screen

- shr()
shift right screen
